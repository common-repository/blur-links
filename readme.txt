== Description ==

This plugin is designed to remove the focus rectangle on a link when clicked. It's very simple and effective.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the 'blur-links' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it! You're done!

== Frequently Asked Questions ==

No.

= Does this plugin make any writes to your plugin database? =

No.

== Screenshots ==

1. In some browsers and older versions of other browsers when you click on a link it puts a focus rectangle around the text or the image of the link.
2. When this plugin is activated it will remove the focus rectangle from the link.
